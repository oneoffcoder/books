# Keras

This directory currently holds Keras-related notebooks under `source/`.

Unlike the main Sphinx books in this repository, this directory does not yet have a local `Makefile`, `pyproject.toml`, or standardized build workflow. Treat it as notebook-backed source material rather than a fully wired Sphinx project.

## Contents

- `source/`: standalone notebooks covering Keras and probabilistic neural network examples

## Working With The Notebooks

Open the notebooks directly in Jupyter Lab or VS Code:

```bash
jupyter lab sphinx/keras/source
```

If this directory is later promoted into a full Sphinx book, add the same per-book build workflow used by the other active books under `sphinx/`.
